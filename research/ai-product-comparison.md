# AI Agent 产品对比 2025

## 市场领导者

### 1. Devin (Cognition)
- **定价**: $500/月 (Devin 2.0: $20/月)
- **收入**: $1M → $73M ARR (9个月增长)
- **特点**: 第一个完全自主的 AI 软件工程师
- **新功能**: Devin Wiki (自动生成文档), Devin Search (代码查询)
- **动作**: 收购 Windsurf IDE

### 2. Claude Code (Anthropic)
- **定价**: $20/月 (API 单独计费)
- **集成**: VS Code, JetBrains, Claude Desktop
- **特点**: 代码理解、多步骤任务、上下文保持
- **新功能**: MCP Tool Search (延迟加载)

### 3. GitHub Copilot
- **定价**: $10/月 (个人), $19/月 (商业)
- **功能**: Agent mode, Coding agent, Agent HQ
- **生态**: 集成多个 provider (Claude/OpenAI/Google)
- **新功能**: MCP 支持, Spec-driven development

### 4. OpenAI Codex CLI
- **定价**: ChatGPT Plus 包含
- **特点**: 轻量级本地运行
- **集成**: VS Code, JetBrains, 终端

### 5. Loki Mode
- **定价**: 开源免费 (MIT)
- **架构**: 37 agents, 6 个群组
- **性能**: HumanEval 98.78%
- **特点**: 完整创业系统 (PRD → 部署)

## 趋势分析

### 商业化加速
- Devin: $73M ARR (最高收入)
- 定价下探: $500 → $20 (Devin 2.0)
- 开源竞争: Loki Mode 免费

### 功能趋同
- 都从「代码补全」→「自主 agent」
- 都支持 MCP 协议
- 都推多 agent 协作

### IDE 整合
- 都推 VS Code 扩展
- Windsurf 被收购 (Devin)
- JetBrains 集成 (Claude/Codex)

## Wayen 机会

### 差异化方向
1. **本地化**: 数据不上云 (如 NanoClaw)
2. **定制化**: 个人专属 agent
3. **轻量级**: 不依赖订阅
4. **多模态**: 不仅代码，还有系统管理

### 可借鉴模式
- Loki Mode: 多 agent 架构
- MCP: 工具标准化
- Devin: 完整生命周期
