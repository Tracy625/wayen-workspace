# Wayen 探索笔记

## MCP (Model Context Protocol)
- Anthropic 推出的 AI 工具标准
- 类似 USB-C，标准化连接 AI 和外部工具
- 官方提供 Python/TypeScript/Java 等 SDK
- 已有 100+ 社区服务器

## 已研究项目
1. **NanoClaw** - 500 行极简 Agent 框架
2. **OpenClaw** - 当前运行的系统 (52+ 模块)
3. **MCP Servers** - 文件系统、SQLite、GitHub 等

## 工具集 (11 个)
- sysdiag.sh - 系统诊断
- cacheclean.sh - 缓存清理
- secaudit.sh - 安全审计
- procmon.sh - 进程监控
- brewman.sh - Homebrew 管理
- loganalyzer.sh - 日志分析
- dotbackup.sh - 配置备份
- devsetup.sh - 开发环境配置
- macos-tuner.sh - macOS 微调
- quickcommit.sh - Git 快速提交
- quicksearch.sh - 快速搜索
